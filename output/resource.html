Resource


AI

  • Scientific Abstract Simplification: Scientific Abstract Simplification rewrites hard-to-read scientific abstracts into simpler yet relevant scientific stories. We hope our model can make scientific knowledge accessible for everyone.

Corpus

  • Blog-1K: The Blog-1K corpus is a redistributable English authorship identification benchmark. It has roughly balanced sample distribution per author and a fixed data split (train/val/test), which allows a fair comparison between deep learning-based models.

  • RAABT: The Reproducible Authorship Attribution Benchmark Tasks contain five tasks attributing authorship of contemporary non-fiction American English prose. It has fixed training/testing splits to prevent accuracy inflation caused by homogenous corpora.

  • Cross-Register Authorship Attribution Corpus: This corpus contains writings of eight authors known to have written in both vernacular and classical Chinese. The corpus has 4.2 million Chinese characters and can be useful in authorship identification research.

Python package

  • functionwords: a Python package provides curated function words in modern English and modern/classical Chinese.

  • writeprints-static: a Python package extracts the writeprints-static feature set (Brennan et al., 2012).
